Monocular Non-rigid Endoscopic 3D Reconstruction

This project presents a monocular endoscopic non-rigid 3D reconstruction method with tissue-aware physical priors and physics-guided depth completion.

Overview

The method integrates non-rigid tracking, heterogeneous physical priors, and diffusion-based dense depth completion for scale-consistent reconstruction.
