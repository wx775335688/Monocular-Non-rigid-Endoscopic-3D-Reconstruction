# Video assets

Put large demo videos in this folder after the repository is created.

Recommended file names:

- `overview_demo.mp4`: short overview of the complete method.
- `tracking_demo.mp4`: front-end non-rigid tracking / sparse reconstruction demo.
- `dense_reconstruction_demo.mp4`: dense depth completion and point-cloud fusion demo.
- `self_collected_scene.mp4`: optional self-collected phantom platform acquisition scene.

After adding videos, edit `docs/index.html` and replace the placeholder in the **Video Demo** section with `<video controls>` blocks.
